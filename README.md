# 🚀 NovitData – Sistem Manajemen Retribusi Sampah

**NovitData** adalah sebuah website yang dirancang untuk membantu pengelolaan retribusi pelayanan angkut sampah di Br. Dangin Jalan secara lebih efisien, terstruktur, dan mudah digunakan.

> ⚠️ **Catatan Penting:**  
> Website ini dibuat khusus untuk penggunaan pribadi (Novita) dan tidak ditujukan untuk akses publik. Sistem ini hanya digunakan oleh pemilik terkait dan bukan untuk penggunaan umum.

---

## 💻 Teknologi yang Digunakan

Website ini dibangun menggunakan:

- HTML → struktur halaman  
- CSS → tampilan dan desain  
- JavaScript → interaksi dan logika sistem  

---

## ✨ Fitur Utama

- 📊 Menampilkan data retribusi warga (Januari – Desember) dalam bentuk tabel  
- ➕ Menambahkan data warga baru  
- ✏️ Mengedit data yang sudah ada  
- 🗑️ Menghapus data dengan konfirmasi  
- 🔍 Pencarian data berdasarkan nama warga  
- 📈 Menampilkan total data secara otomatis  
- 📄 Pagination untuk navigasi data  
- 📥 Export data ke Excel  

---

## 📦 Library yang Digunakan

- GSAP (GreenSock Animation Platform) → animasi antarmuka  
- SheetJS (xlsx) → export data ke Excel  

---

## 🎯 Tujuan Pengembangan

Website ini dibuat untuk memberikan solusi digital dalam pengelolaan data retribusi agar lebih:

- Praktis  
- Cepat  
- Minim kesalahan
